🔧 Server Testing Commands

Use these commands to test different server features:

# Test basic GET request curl -v http://127.0.0.1:8080/ # Test missing Host header (should return 400) curl -v -H "Host:" http://127.0.0.1:8080/ # Test path traversal (should return 403) curl -v http://127.0.0.1:8080/../etc/passwd # Test unsupported method (should return 405) curl -v -X PUT http://127.0.0.1:8080/ # Test JSON upload curl -v -X POST -H "Content-Type: application/json" \ -d '{"name":"Test","message":"Hello"}' \ http://127.0.0.1:8080/upload # Test keep-alive connection curl -v --keepalive http://127.0.0.1:8080/
