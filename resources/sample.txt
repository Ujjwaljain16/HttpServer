The server handles various file types:

- Text files (.txt) - served as application/octet-stream with attachment
- Image files (.png, .jpg, .jpeg) - served as application/octet-stream with attachment
- Other files - return 415 Unsupported Media Type

- Path traversal protection (blocks ../ attacks)
- Host header validation
- Input validation for JSON uploads
- Comprehensive error handling

- Thread pool with configurable worker count
- Bounded queue for request handling
- Keep-alive connections with 30s timeout
- Maximum 100 requests per connection
